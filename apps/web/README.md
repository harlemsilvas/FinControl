# FinControl Web

Fundação React do frontend FinControl.

```bash
npm run dev --workspace @fincontrol/web
npm run test --workspace @fincontrol/web
npm run build --workspace @fincontrol/web
```

A URL da API é configurada por `VITE_API_URL`. A Fase 9 contém apenas a fundação;
login e telas de cadastros pertencem à Fase 10.
